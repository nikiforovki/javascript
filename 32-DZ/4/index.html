Профиль пользователя

Разница между PUT и PATCH запросами

PUT - полностью заменяет ресурс новыми данными. Требует отправки полного представления ресурса.

PATCH - вносит частичные изменения в ресурс. Отправляются только измененные поля.

Пример:

PUT /users/1 {name: "Иван", email: "ivan@example.com"}

→ Все поля пользователя будут заменены

PATCH /users/1 {email: "new@example.com"}

→ Изменится только email, остальные поля останутся прежними
